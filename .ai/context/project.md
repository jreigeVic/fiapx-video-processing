# Project Overview

## Nome

FIAP X - Cloud Native Video Processing Platform

## Objetivo

Construir uma plataforma cloud-native para processamento assíncrono de vídeos utilizando microsserviços, arquitetura orientada a eventos e infraestrutura Kubernetes.

O projeto faz parte do Hackathon da Pós-Tech FIAP.

---

## Escopo

O sistema permite:

- autenticação de usuários
- upload de vídeos
- processamento assíncrono
- extração de frames
- geração de ZIP
- download
- consulta de status
- observabilidade
- escalabilidade

---

## Fora do escopo

- Frontend completo
- Analytics
- Painel administrativo
- Dashboard de negócio

---

## Objetivo técnico

Demonstrar conhecimentos de:

- arquitetura
- microsserviços
- cloud native
- kubernetes
- mensageria
- observabilidade
- CI/CD
