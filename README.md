# FIAP X - Cloud Native Video Processing Platform

Cloud-native video processing platform developed for the FIAP X Hackathon.

## Goals

- Cloud Native Architecture
- Event-Driven Architecture
- Microservices
- Kubernetes
- AWS SNS/SQS/S3
- CI/CD
- Observability
- AI-Assisted Development

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- AWS SNS
- AWS SQS
- AWS S3
- Kubernetes
- Terraform
- GitHub Actions
- OpenTelemetry
- New Relic

## Architecture

- Clean Architecture
- Hexagonal Architecture
- Domain-Driven Design (Lite)
- Event-Driven Architecture

> This repository follows an AI-first engineering workflow where architectural decisions are made by engineers and repetitive implementation is accelerated through AI.
