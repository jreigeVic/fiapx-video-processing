# High Level Design (HLD)

## Objetivo

Este diretório contém a documentaçăo arquitetural de alto nível da plataforma **FIAP X Video Processing**.

O High Level Design (HLD) descreve a soluçăo sob uma perspectiva arquitetural, apresentando a visăo macro do sistema, seus componentes, principais decisőes técnicas e como os requisitos funcionais e năo funcionais săo atendidos.

Esta documentaçăo serve como referęncia para:

- Arquitetos de Software
- Desenvolvedores
- DevOps Engineers
- QA Engineers
- Equipe de Operaçőes
- Ferramentas de IA utilizadas durante o desenvolvimento

# Relaçăo com o LLD

O HLD descreve **o que** será construído.

O LLD descreve **como** cada componente será implementado.

Todo documento do LLD deve possuir rastreabilidade para esta documentaçăo.
